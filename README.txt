Currently supports the following devices
- HS1SA-Z - Smart Smoke Sensor
- HS1CA-Z - Smart Carbon Monoxide Sensor
- HS1CG-Z - Smart Carbon Monoxide Sensor
- HS1HT-Z - Smart Temperature & Humidity Sensor
- HS1MS-Z - Smart Motion Sensor
- HS2SKm-Z - Smart Metering Plug
- HS2SK-Z - Smart Plug
- HS2WD-Z - Smart Sound and Flash Siren
